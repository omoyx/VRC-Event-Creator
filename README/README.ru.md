Универсальный инструмент создания событий для VRChat, который избавляет от повторяющейся настройки.
Создавайте и сохраняйте шаблоны событий для каждой группы, генерируйте ближайшие даты из простых повторяющихся паттернов и мгновенно заполняйте детали - идеально для быстрого планирования еженедельных встреч, просмотровых вечеров и событий сообщества.

## Возможности
- Профили/шаблоны, которые автоматически заполняют детали события для каждой группы.
- Генератор повторяющихся шаблонов со списком ближайших дат и ручным вводом даты/времени.
- Система автоматизации событий (экспериментальная) - автоматическая публикация событий на основе шаблонов профилей.
- Мастер создания событий для групповых календарей.
- Раздел «Редактировать события» для будущих событий (сетка + модальное окно редактирования).
- Студия тем с пресетами и полным контролем цветов UI (поддержка #RRGGBBAA).
- Выбор и загрузка изображений галереи для ID изображений.
- Сворачивание в системный трей.
- Локализация с выбором языка при первом запуске (en, fr, es, de, ja, zh, pt, ko, ru).

## Конфиденциальность и хранение данных
Ваш пароль не хранится. Кэшируются только токены сессии.
Приложение хранит файлы в каталоге данных пользователя Electron (см. Настройки > Информация о приложении):

- `profiles.json` (шаблоны профилей)
- `cache.json` (токены сессии)
- `settings.json` (настройки приложения)
- `themes.json` (пресеты темы и пользовательские цвета)
- `pending-events.json` (очередь автоматизации)
- `automation-state.json` (отслеживание автоматизации)

Каталог данных можно переопределить переменной окружения `VRC_EVENT_DATA_DIR`.
При первом запуске приложение попытается импортировать существующий `profiles.json` из папки проекта.

__**Не делитесь файлами кэша или папками данных приложения.**__

## Примечания по использованию
- Профили требуют название профиля, название события и описание перед продолжением.
- Для приватных групп доступен только тип доступа = Группа.
- Длительность в формате DD:HH:MM и ограничена 31 днём.
- Теги ограничены 5, языки — 3.
- Загрузка в галерею: PNG/JPG, 64-2048 px, меньше 10 МБ и 64 изображения на аккаунт.
- VRChat ограничивает создание событий до 10 событий в час на человека на группу.
- Автоматизация событий требует, чтобы приложение было запущено. Пропущенными автоматизациями можно управлять в разделе Редактировать события.

## Устранение неполадок
- Проблемы входа: удалите `cache.json` и войдите снова (используйте папку данных из Настройки > Информация о приложении).
- Нет групп: учетной записи нужен доступ к календарю целевой группы.
- Ограничение скорости: VRChat может ограничивать создание событий. Подождите и попробуйте снова, и остановитесь, если несколько попыток неудачны. Не спамьте кнопки обновления или создания событий.
- Обновления: Некоторые функции блокируются, когда есть ожидающие обновления. Скачайте и запустите последнюю версию.
